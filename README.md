Face Detection App
==============================


### A React app that Detects human face.
## Features
- ##### A Login and resgistration Page.
- ##### A Form page that takes a image URL an input.
- ##### Strats implementing face-detection API FROM [Clarifai].
- ##### Draws boxes around detected faces.
